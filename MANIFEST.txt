INSTALLATION QUICK REFERENCE:
================================================================================

1. EXTRACT TO XAMPP:
   C:\xampp\htdocs\child-learning-system\

2. CREATE DATABASE:
   - Open phpMyAdmin (http://localhost/phpmyadmin)
   - Run database.sql
   - Optional: Run seed_data.sql for demo data

3. START SERVICES:
   - Start Apache in XAMPP
   - Start MySQL in XAMPP

4. ACCESS APPLICATION:
   - Go to http://localhost/child-learning-system/
   - Login with demo credentials above

5. CREATE REAL DATA:
   - Login as admin
   - Create users, classes, students
   - Assign teachers to classes
   - Login as teacher to create activities
   - Login as student to use activities
