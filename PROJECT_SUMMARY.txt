QUICK START STEPS:
================================================================================

1. Extract files to: C:\xampp\htdocs\child-learning-system\

2. Create database:
   - Open phpMyAdmin
   - Run database.sql
   - (Optional) Run seed_data.sql for sample data

3. Start XAMPP:
   - Start Apache
   - Start MySQL

4. Access application:
   - Navigate to http://localhost/child-learning-system/
   - Login with demo credentials above

5. Create your own data:
   - Login as admin
   - Create users, classes, students
   - Login as teacher to create activities
   - Login as student to attempt activities
